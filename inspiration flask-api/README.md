# RCM Analysis API

This project exposes a Flask API for four RCM tools:

1. `POST/GET /api/ai-suggest`
2. `POST/GET /api/control-assessment`
3. `POST/GET /api/deduplication`
4. `POST/GET /api/tod-test`

Start server:

```bash
python api.py
```

Base URL:

```text
http://localhost:5002
```

For full request/response examples, see `README_API.md`.
